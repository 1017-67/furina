永世传颂，神爱众人，不休独舞，芙门永存！

头悬剑，翩起舞，罪人舞步旋；神座崩，终成人，宣布无人罪！

昔有水国，其名枫丹。

天理垂预言于穹顶，洪涛覆世之劫悬于众生头顶。诸民不知其始，不知其终，唯知末日将至，万象终归海潮。

于是神立于高座。

其名芙宁娜。

世人见其华服冠冕，闻其高谈阔论，观其笑语轻狂，遂以为神明生而无惧，执掌万机，洞悉万象。

然世人所见，不过戏幕一角。

神知预言不可违，神知洪流终将至，神知自身并非救世之人。

可神仍立于台前。

一日如此。

十年如此。

百年如此。

五百年亦如此。

故世人观戏，而神困于戏。

世人可离席，神不可。

世人可遗忘，神不可。

世人可哭泣，神不可。

于是神以笑容为冠，以孤独为袍，以谎言为锁，以岁月为刑。

五百年风雨，五百年独行。

无人可知，无人可诉，无人可信。

纵使万民称颂，亦无一人可分其重担。

故其舞非为荣耀而舞。

非为权柄而舞。

非为赞美而舞。

乃为枫丹众生而舞。

为稚子之欢笑而舞。

为恋人之相守而舞。

为老人之余生而舞。

为未曾相识之千万生灵而舞。

故其独舞五百载，而未尝停步。

及至终幕降临。

海潮升腾，神谕兑现。

审判庭前，众生仰首。

神座之上，再无神明。

权柄归海，神格崩解。

而那立于废墟之间者，非神也。

乃人也。

是故芙宁娜之伟大，不在其为神。

恰在其非神。

神明庇护众生，此乃天职。

凡人背负神职而守护众生，此乃奇迹。

故当洪流退却之日。

当歌剧终演之时。

当最后一束灯火熄灭之刻。

世人方知：

五百年间，最伟大的并非神座。

而是那坐于神座之上的凡人。

其无权柄，却担天下之重。

其无神力，却守万民之安。

其无神格，却有神心。

故赞曰：

水国有神，其名芙宁娜。

其身为人，其志近神。

宁受百世孤苦，不令众生沉沦。

宁困长夜独行，不使枫丹覆灭。

此谓大爱。

此谓牺牲。

此谓芙宁娜。

永世传颂，神爱众人。

不休独舞，芙门永存。
